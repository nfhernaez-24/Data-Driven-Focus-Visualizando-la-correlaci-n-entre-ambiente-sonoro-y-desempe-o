# 🎧 Análisis de Eficacia Cognitiva y Entorno Sonoro

## 📌 Descripción del Proyecto
Este dashboard interactivo en **Power BI** analiza cómo diferentes entornos sonoros (Silencio, Café, Música, Tráfico, etc.) afectan la **Eficacia Cognitiva** y el **Costo de Fatiga** en distintos perfiles: Profesionales, Trabajadores Remotos y Estudiantes.

El objetivo es identificar patrones de enfoque y agotamiento mental según la tarea realizada (Coding, Reading, Studying, Writing), permitiendo optimizar los entornos de trabajo para mejorar el rendimiento.

## 📊 KPIs Principales
* **Eficacia Cognitiva:** Medida del rendimiento promedio por tarea.
* **Costo de Fatiga:** Índice de agotamiento acumulado según el entorno.
* **Puntuación de Enfoque:** Nivel de concentración reportado/medido por tipo de ruido.

## 🧠 Insights Destacados
* **Eficacia por Tarea:** Se observa una correlación directa entre el tipo de ruido y la tarea; por ejemplo, el *Coding* presenta picos de eficacia bajo ciertos sonidos específicos en comparación con la lectura.
* **Perfil del Trabajador Remoto:** El análisis segmentado muestra cómo la fatiga mental impacta de manera distinta a quienes trabajan desde casa frente a estudiantes.
* **Impacto del Ruido:** El gráfico de "Enfoque por tipo de ruido" revela que sonidos como *Songs with Lyrics* tienen un impacto significativo en la dispersión comparado con *Instrumental Music*.

## 🛠️ Herramientas Utilizadas
* **Power BI Desktop:** Modelado de datos y visualización.
* **Power Query:** Limpieza y transformación de la data (ETL).
* **DAX:** Creación de medidas personalizadas para el cálculo de Eficacia y Costos de Fatiga.
* **Diseño:** UI/UX enfocado en modo oscuro para reducir la fatiga visual del usuario.
